# Maintenance and Safety Module

To ensure the long-term reliability and safety of the industrial bakfiets under maximum cargo loads, a strict maintenance and operational protocol is required.

## Safety Protocols
### 1. Heavy Cargo Handling
- **Load Distribution**: Always place the heaviest items at the bottom and center of the cargo box.
- **Stabilization**: Engage all four corner stabilizer feet before loading or unloading using a forklift.
- **Maximum Load**: 350kg (Total vehicle + cargo + operator should not exceed 500kg).

### 2. Operational Safety
- **Startup**: Always power on the master controller first and verify brake cutoff functionality before motion.
- **Braking**: Rely on the front 203mm hydraulic brake as the primary stopping power, using the rear 180mm brake for control.
- **Speed**: Do not exceed 25km/h when fully loaded.

## Preventative Maintenance (PM) Schedule
| Interval | Component | Action |
| --- | --- | --- |
| **Daily** | Brakes | Test lever pressure and motor cutoff switches. |
| | Tires | Inspect for punctures and check pressure (3.5-4.0 bar). |
| | Battery | Check state of charge and ensure equalizer connection is tight. |
| **Weekly** | Fasteners | Check torque on M14 axle nuts and M8 rim bolts. |
| | Chain | Clean and lubricate industrial-grade chain. |
| | Cables | Inspect corrugated tubing and spiral wraps for rub points. |
| **Monthly** | Steering | Check M10 rod ends for play and lubricate if necessary. |
| | Controllers | Inspect housing boxes for moisture ingress. |
| **Quarterly** | Bearings | Inspect 7202 steering bearings and 6202 hub bearings for smoothness. |
| | Frame | Inspect weld points on main spars and cargo bed for stress cracks. |

## Build Checklist (Final Verification)
- [ ] All fasteners are Grade 8.8 or higher and tightened to specified torque.
- [ ] Stabilizer legs extend and retract smoothly with a cordless drill.
- [ ] 13\" car tire is correctly seated in the split rim.
- [ ] Wiring is fully protected in PA6 conduit and routed away from the chain line.
- [ ] PAS sensor triggers rear motor and throttle triggers both motors.
