# Maintenance and Fleet Management Log

The Industrial E-Bakfiets is designed for heavy usage. This log provides a framework for periodic inspections and preventative maintenance.

## 1. Maintenance Schedule

| Interval | Component | Action | Specification |
| --- | --- | --- | --- |
| **Daily** | Brakes | Check lever feel and pad contact | 203mm rotor check |
| | Tires | Check pressure (Front: 45psi, Rear: 35psi) | Visual check for cuts |
| | Battery | Charge to 100% and balance | 54.6V (48V pack) |
| **Weekly** | Chain | Clean and lubricate | Industrial wet-lube |
| | Fasteners | Torque check Axle Nuts and Rim Bolts | M14: 50Nm, M8: 30Nm |
| | Steering | Check for play in rod ends | Zero play required |
| **Monthly** | Bearings | Inspect 7202 Steering bearings | Re-grease if gritty |
| | Spokes | Pluck test for tension (Rear 16") | 10G heavy spokes |
| | Wiring | Check for insulation abrasion | Inspect PA6 conduits |
| **6-Months** | Hub Motors | Check for moisture/rust on flanges | Apply anti-corrosive |
| | Brake Fluid | Check level and color | Mineral Oil / DOT 4 |

## 3. Industrial Spare Parts Kit
It is recommended for fleet operators to keep the following in stock:
- 2x M8x140 Grade 10.9 Rim Bolts.
- 1x Set of 203mm Brake Pads.
- 1x Spare 13-inch Car Inner Tube.
- 1x M10 LH/RH Rod End set.
- 1x Pair of 7202 Angular Contact Bearings.
