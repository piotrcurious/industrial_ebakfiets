# Cargo Box Module

The cargo box of the industrial bakfiets is designed for maximum utility, standard freight compatibility, and extreme durability.

## Construction Materials
### 1. Main Panels
- **Specification**: **12mm Marine-grade Plywood**.

### 2. Reinforcements
- **Edge Protection**: **25mm x 25mm x 3mm Steel Angle**.

## Modular Divider System
To prevent cargo movement during transit, the box is equipped with an adjustable partitioning system.

### 1. Logistics Rails
- **Standard**: **50mm E-track Rails** (Zinc-plated steel).
- **Mounting**: Bolted to the internal horizontal mid-line of the box using M6 countersunk bolts.

### 2. Partition Panels
- **Material**: 9mm Phenolic plywood.
- **Hardware**: Integrated **E-track Spring Fittings** allow the panels to be clicked into place at any point along the rails.

## Dimensions and Compatibility
- **Internal Width**: 810mm.
- **Internal Length**: 1350mm.
- **Standard Load**: Fits one **Standard Euro-pallet** (with extra room for gear).

## Mounting and Security
- **Frame Interface**: Secured via **8x M10 Grade 8.8 Bolts**.
- **Tie-Down Points**: Four integrated **D-ring anchor points**.
