# Element: Rear Frame Dropout Plate

The rear dropouts accommodate the 135mm hub motor and the 7-10 speed derailleur hanger.

## Technical Specifications
- **Material**: 8.0mm Structural Mild Steel.
- **Manufacturing**: Laser-cut.

## Geometry
- **Axle Slot**: 10.2mm (standard rear hub axle).
- **Derailleur Hanger**: Integrated tab with M10x1.0 threaded hole, offset 10mm outward.
- **Brake Mount**: IS (International Standard) tabs.

## Installation Notes
- Weld to the ends of the tapered rear triangle tubes.
- Use a 135mm precision spacer during welding to prevent the "pulling" of the dropouts as the welds cool.
